Rental Management and Website Rental Product - Odoo App

Product/Item/Equipment/Vehicle Rental Management

Odoo App for Rental Products Management

This app allow you to manage rental of product / items.

This app will work for all kind of Rental business/Industry/Shop. You can use this app to rent any kind of Products/Items, Home, Office, Car, Bike, Service on Rent, AC, TV, Office Rent, and many more...

Main Features:

  • Allow Sales Team to create Rental Products in Odoo. Please note that rental product is same as Sales/Normal Product of Odoo so we are using Product model of Odoo only (Technically).
  • It will show product rental availiblity on rental request page for rental products to your customer once they choose start and end dates.
  • Customer can buy rental products directly from Rental request menu on web
  • Allow your customers to select start datetime and end datetime for rent of that product.
  • Automatic confirm order if submit from rental request form else sales team can confirm rental order manually as well from backend.
  • Sales order PDF report with start and end datetime of rental orders.
  • Allow you to configure rental prices on product form. This will give you feature where you can set rent price / day based on rental periods/count, pricelist and variants.
  • Only Visible products on rental request form which is allow for rent and published for rent
  • Customer can choose now start and end datetime of rental period and system will show availibility and prices accordingly.
  • Sales order will keep information of rental.
  • System will show status/warning message in rental request web on website product it will forcefully restrict customer if product is not available. System is expecting that customer will see message and get to know status and then can choose other period or will skip that product.
  • System is respecting rental quantity and reservations and based on that is show status to customer.
